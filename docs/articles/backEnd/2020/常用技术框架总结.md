存储使用  `Minio / OSS`

权限管理  `Shiro  / jwt + security`

搜索管理 `Elasticsearch` 配合分词 `HanLP`

数据请求 `WebClient` (`RestTemplate` `Spring` 团队已不建议使用, `Webclient` 中进行同步处理 `.block()` )

数据延迟 `Rabbitmq`

数据库交互 ` MyBaits  / MyBaits Plus`

数据库交互监控 `Druid`

数据接口文档  `Swagger UI`
